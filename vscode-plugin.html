◈ VSCode Extension

CiteCheck brings citation validation directly into your LaTeX/BibTeX workflow in VS Code. Get inline diagnostics, hover tooltips on \cite{} keys, real-time validation, and a dedicated results panel — without leaving your editor.

Features

Inline DiagnosticsRed/yellow squiggles on unverified or fake citations in .bib files
Hover TooltipsHover over @key to see CrossRef match status, DOI, and confidence
Validate on SaveAutomatically checks citations when you save a .bib file
Status BarLive count of verified / fake / partial citations in the status bar
Results WebviewFull results panel with all citations, scores, and CrossRef links
Workspace ScanValidate all .bib files in your project at once

Installation (Marketplace)

1
Open VS Code → Extensions (Ctrl+Shift+X)
2
Search for "CiteCheck" by kavihai
3
Click Install. No restart required.
4
Open any .bib file — validation starts automatically.

Installation (Manual / Dev)

# Clone the repo git clone https://github.com/kavihai/citecheck-vscode cd citecheck-vscode # Install dependencies npm install # Package the extension npx vsce package # Install the .vsix file code --install-extension citecheck-1.0.0.vsix

Commands

Ctrl+Shift+P → CiteCheck: Validate Current .bib File Ctrl+Shift+P → CiteCheck: Validate Selected Citation Ctrl+Shift+P → CiteCheck: Validate All .bib Files in Workspace Ctrl+Shift+P → CiteCheck: Show Results Panel # Right-click menu in .bib files: → CiteCheck: Validate Current .bib File → CiteCheck: Validate Selected Citation

Extension Architecture

vscode-extension/ ├── package.json ← Extension manifest, commands, settings ├── src/ │ └── extension.js ← Main extension logic │ ├── activate() ← Registration of commands, hover, events │ ├── validateDocument() ← Parses .bib → CrossRef → diagnostics │ ├── parseBib() ← BibTeX AST parser with Range tracking │ ├── validateOne() ← CrossRef DOI / title lookup │ ├── scoreMatch() ← Confidence scoring │ ├── provideHover() ← Hover tooltip provider │ ├── updateDecorations() ← Inline ✓/✗ icons │ └── showResultsPanel() ← Webview HTML panel

How It Works

→
On file open/save, parseBib() extracts all @article{} entries with their source ranges.
→
Each citation is checked via CrossRef DOI lookup first, then title/author search as fallback.
→
Confidence is scored by title similarity + year match. Threshold defaults to 80%.
→
Diagnostics (red/yellow squiggles) are applied to the exact source range of each entry.
→
Status bar updates to show: $(check-all) CiteCheck: 12 OK | 2 fake

Requirements

VS Code 1.85+ · Node.js 18+ (for development) · Internet connection · No API key required
