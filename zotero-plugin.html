⬡ Zotero Plugin

CiteCheck integrates directly with your Zotero reference manager. Validate citations with a right-click, auto-tag suspicious entries, and batch-validate entire collections — all powered by CrossRef's 150M+ publication database.

Features

Right-Click ValidationSelect items → Tools → CiteCheck → Validate
Auto-TaggingTags items as "CiteCheck: Verified", "CiteCheck: Not Found" etc.
Batch ValidationValidate your entire library or a specific collection at once
CrossRef VerifiedReal-time lookup against 150M+ academic publications
Zotero 6 & 7Compatible with both major Zotero versions
Progress WindowLive progress with per-item status updates

Installation

1
Download the citecheck-zotero.xpi file below.
2
Open Zotero → Tools → Add-ons.
3
Click the gear icon ⚙ → Install Add-on From File…
4
Select the downloaded .xpi file and click Install Now.
5
Restart Zotero. CiteCheck will appear under Tools menu.
⬇ Download Plugin Files (ZIP)

Usage

→
Select one or more items in your Zotero library.
→
Go to Tools → CiteCheck → Validate Selected Citations.
→
A progress window shows each citation being checked.
→
Results dialog shows summary. Items are auto-tagged in your library.

Tags Applied

CiteCheck: Verified CiteCheck: Not Found CiteCheck: Partial Match CiteCheck: Unknown
