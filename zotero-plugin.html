(fig. 01) — Features

CiteCheck integrates directly with your Zotero reference manager. Validate citations with a right-click, auto-tag suspicious entries, and batch-validate entire collections — all without leaving your workflow.

Right-Click Validation Select items → Tools → CiteCheck → Validate. No window-switching required.
Auto-Tagging Items are automatically tagged as Verified, Not Found, Partial Match, or Unknown.
Batch Validation Validate your entire library or a specific collection in a single operation.
CrossRef Verified Real-time lookup against 150M+ academic publications via the polite CrossRef API.
Zotero 6 & 7 Support Compatible with both major Zotero versions via the bootstrap plugin architecture.
Progress Window Live progress dialog with per-item status updates as validation runs.
(fig. 02) — Installation

Installing the plugin.

1
Download the citecheck-zotero.xpi file below.
2
Open Zotero → Tools → Add-ons.
3
Click the gear icon ⚙ → Install Add-on From File…
4
Select the downloaded .xpi file and click Install Now.
5
Restart Zotero. CiteCheck appears under the Tools menu.

(fig. 03) — Usage

Running a validation.

→
Select one or more items in your Zotero library.
→
Go to Tools → CiteCheck → Validate Selected Citations.
→
A progress window shows each citation being checked against CrossRef.
→
Results dialog shows a summary. Items are auto-tagged in your library.
Tags applied to items
CiteCheck: Verified CiteCheck: Not Found CiteCheck: Partial Match CiteCheck: Unknown
(fig. 04) — Architecture

Plugin source code.

// bootstrap.js — Main plugin entry point // Located at: zotero-plugin/src/bootstrap.js CiteCheck.validateSelected() → parsedItems: Zotero.getActiveZoteroPane().getSelectedItems() → CrossRef DOI lookup → title search fallback → Tags items with validation status → Shows results dialog // Auto-tagging example: item.addTag('CiteCheck: Verified', 1); // type 1 = automatic await item.saveTx();
